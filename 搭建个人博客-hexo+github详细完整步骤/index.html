一、第一步：下载暗转Git

1.Git 下载地址

https://gitforwindows.org/

2.安装步骤

无标题.png

无标题.png

剩下的一路默认

安装路径根据自己需求

二、第二步：下载安装node.js

1.node.js下载地址

https://nodejs.org/en/

image.png

2. 安装步骤

默认就行

安装路径根据自己需求

第三步：安装Hexo

1.利用npm命令安装

任意位置鼠标右键选中Git Bash Here

image.png

2.输入命令： npm install -g hexo

-g 是全局的意思

如安装失败，请更换网络

image.png

第四步：初始化Hexo

1.创建文件夹

命名规则无强制要求

image.png

2.hexo init（Hexo本地初始化）

在Hexo文件下，右键运行Git Bash，输入命令：hexo init

安装比较久的，请更换网络

image.png

成功后默认生成

image.png

3. 配置_config.yml

在_config.yml 进行简单的配置

image.png

4.本地浏览博客

在Hexo文件夹中打开Git Bash 分别输入

hexo g 生成网站静态文件到默认设置的 public 文件夹

hexo s 开启服务

image.png

在浏览器输入：localhost：4000 ，就可以进行访问，效果如下：

image.png

5.写文章

在F:\Hexo\source_posts文件下，新建.md文件就可以写文章

image.png

第五步：部署到Github上

1.申请Github账户

2.新建一个仓库

new repository

image.png

仓库名字必需跟前面保持一致，固定格式image.png

然后点击创建，其他默认

image.png

3.在_config.yml进行配置

添加仓库地址image.png

4.安装hexo-deployer-git自动部署发布工具

hexo-deployer-git仓库地址：

https://github.com/hexojs/hexo-deployer-git

用部署工具可以更快捷的push你的网站，不用像以前一样还要配置SHH

npm install hexo-deployer-git –save

image.png

5.发布到Github

输入如下命令：hexo clean && hexo g && hexo d

image.png

image.png

登录后授权就行了

如第一次没有出现登录，检查是否配置Gti User信息，网络问题请更换网络

发布完成

image.png

image.png

6.测试访问

在浏览器输入：https://hh9612.github.io/

image.png
